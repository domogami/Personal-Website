# My Old Website
My old website was built with Gatsby and hosted with Netlify. To build locally, clone the repo

```bash
git clone https://github.com/domogami/Personal-Website.git
cd Personal-Website
```

And to develop locally, run the following

```bash
npm install
gatsby develop
```
